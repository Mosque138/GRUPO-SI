1.  qué problema soluciona?
Rappi soluciona el problema de la falta de tiempo y movilidad de las personas para comprar productos o solicitar algún servicio. Permite pedir desde el celular comida, medicamentos y otros productos y recibirlos en casa de forma rápida

2.  a qué nivel de decisión impacta?
Impacta más que todo en el nivel operativo porque el sistema de rappi se utiliza para gestionar las actividades diarias como la recepción de pedidos, asignación de repartidores, seguimiento de entregas y tiempo de servicio

3. que decisiones permitirá tomar?
- Decir que productos tienen mayor demanda
- Ajustar precios y promociones
- Mejorar tiempos de servicio 

4. que pasaría si no existiera ese sistema?
- Las personas tendrían que desplazarse físicamente para comprar
- Los negocios venderían menos
- Se perdería tiempo en filas y transporte
